# ChatFlow – Dynamic Messaging System

A dynamic messaging platform built using Node.js, Express, EJS, and MongoDB.

## 🚀 Features
- Create, edit, delete messages (CRUD)
- Dynamic UI using EJS templates
- RESTful routing
- Backend with Express.js

## 🛠 Tech Stack
- Node.js
- Express.js
- EJS
- MongoDB
- CSS

## ▶️ How to Run
1. Clone the repository
2. Run `npm install`
3. Start server using `node app.js`
4. Open in browser: http://localhost:8080

## 📌 Project Purpose
This project demonstrates backend development, routing, and database integration.
